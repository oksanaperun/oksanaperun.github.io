Пошук слів

У формі пошуку можна використовувати наступні символи:

  • * (зірочка) - для позначення будь-якої кількості літер, в тому числі ні однієї
  • . (крапка) - для позначення будь-якої рівно однієї літери
  • ' (апостроф) - для позначення апострофа
