Пошук слів

У формі пошуку можна використовувати наступні символи:

* (зірочка) - для позначення будь-якої кількості літер, в тому числі ні однієї

. (крапка) - для позначення будь-якої рівно однієї літери

' (апостроф) - для позначення апострофа

1. Щоб дізнатися, чи існує певне слово, просто вводимо його в поле пошуку, наприклад, "бук"

2. Щоб знайти всі слова, які розпочинаються на "бук", вводимо в поле пошуку "бук*"

3. Щоб знайти всі слова, які закінчуються на "бук", вводимо в поле пошуку "*бук"

4. Щоб знайти всі слова, які містять в собі "бук", вводимо в поле пошуку "*бук*"

5. Щоб знайти всі слова, які складаються з трьох літер, вводимо в поле пошуку "..."

6. Щоб знайти всі слова, які складаються з п'яти літер і розпочинаються на "бук", вводимо в поле пошуку "бук.."

7. Щоб знайти всі слова, які містять апостроф, вводимо в поле пошуку "*'*"
